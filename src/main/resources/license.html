In dieser Software werden folgende Datensätze von NVBW Open Data und MobiData BW eingesetzt:

Datensatz Barrierefreiheit an BW-Bahnhöfen
Datensatz Haltestellen

Diese Daten sind vom Datengeber mit einer Datenlizenz Deutschland – Namensnennung – Version 2.0 lizenziert:
Lizenz Datensatz Barrierefreiheit an BW-Bahnhöfen
Lizenz Datensatz Haltestellen

Der Datensatz Barrierefreiheit an BW-Bahnhöfen wird beim Start der Software komplett vom Server heruntergeladen - Bilder werden dynamisch nachgeladen.

Der Datensatz Haltestellen ist in einer abgespeckten Version (nicht benötigte Daten entfernt) im Quelltext der Software hinterlegt, um den Bahnhöfen den Ortsnamen und den Landkreis zuordnen zu können.
