Diese Daten sind vom Datengeber mit einer Datenlizenz Deutschland – Namensnennung – Version 2.0 lizenziert:
Lizenz Datensatz Barrierefreiheit an BW-Haltestellen
Lizenz Datensatz Haltestellen

Der Datensatz Barrierefreiheit an BW-Haltestellen wird beim Start der Software komplett vom Server heruntergeladen - Bilder werden dynamisch nachgeladen.

Der Datensatz Haltestellen ist in einer abgespeckten Version (nicht benötigte Daten entfernt) im Quelltext der Software hinterlegt, um den Bahnhöfen Ortsnamen und Landkreis zuordnen zu können.
