In dieser Software verwendete Daten von NVBW Open Data:

Datensatz Barrierefreiheit an BW-Haltestellen
Haltestellen

Diese Daten sind von NVBW mit folgender Lizenz lizenziert:
Lizenz Datensatz Barrierefreiheit an BW-Haltestellen
Lizenz Datensatz Haltestellen

Der Datensatz Barrierefreiheit an BW-Haltestellen wird beim Start der Software komplett vom Server heruntergeladen - Bilder werden dynamisch nachgeladen.

Der Datensatz Haltestellen ist in einer abgespeckten Version im Quelltext der Software hinterlegt, um Ortsnamen zuordnen zu können.
